# Elastic-py3

## Dowmload
```bash
git clone https://github.com/hdb106/Elastic-py3.git
```

## Unzip
```bash
cd Elastic-py3
tar -zxvf ElaStic.tar.gz
```

## Configure environment variables
```bash
export ElaSticROOT=/home/$USER/software/ElaStic_1.1
export PATH=/home/$USER /software/ElaStic_1.1:$PATH
```
